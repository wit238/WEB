คำอธิบาย: body { position: relative; min-height: 100vh; }: กำหนดให้ body เป็น container สำหรับ position: absolute และให้มีความสูงเต็ม viewport footer { position: absolute; bottom: 0; left: 0; width: 100%; }: กำหนดให้ footer อยู่ด้านล่างสุดของ body โดยใช้ position: absolute ข้อควรจำ: เลือกใช้วิธี Flexbox หรือ CSS Grid เนื่องจากเป็นวิธีที่ทันสมัยและยืดหยุ่นกว่า ตรวจสอบให้แน่ใจว่าได้กำหนด min-height: 100vh; ให้กับ body เพื่อให้ footer ติดด้านล่างแม้ว่าเนื้อหาในหน้าเว็บจะมีน้อย ปรับแต่งสไตล์ของ footer ตามต้องการ col-md-6 คืออะไร col-md-6 เป็นคลาสของ Bootstrap ที่ใช้ในการจัดวางองค์ประกอบภายในระบบ Grid System ของ Bootstrap โดยมีความหมายดังนี้: col-: หมายถึงคลาสที่ใช้สำหรับกำหนดคอลัมน์ในระบบ Grid md-: หมายถึง breakpoint หรือขนาดหน้าจอระดับ medium (ขนาดกลาง) ซึ่งโดยทั่วไปคือหน้าจอที่มีความกว้างตั้งแต่ 768 พิกเซลขึ้นไป เช่น แท็บเล็ตแนวนอน หรือหน้าจอคอมพิวเตอร์ขนาดเล็ก 6: หมายถึงขนาดของคอลัมน์ โดย Bootstrap แบ่งหน้าจอออกเป็น 12 ส่วนเท่าๆ กัน ดังนั้น col-md-6 หมายถึงคอลัมน์นี้จะมีความกว้าง 6 ส่วนจาก 12 ส่วน เมื่อแสดงผลบนหน้าจอขนาด medium ขึ้นไป สรุป: col-md-6 จะทำให้องค์ประกอบนั้นกินพื้นที่ครึ่งหนึ่งของแถว (row) เมื่อแสดงผลบนหน้าจอขนาดกลาง (md) ขึ้นไป หากหน้าจอมีขนาดเล็กกว่า medium (เช่น หน้าจอมือถือ) คอลัมน์อาจแสดงผลแบบเต็มความกว้างของแถว หรือแสดงผลตามคลาสอื่นๆ ที่กำหนดไว้สำหรับขนาดหน้าจอที่เล็กกว่า ระบบ Grid ของ Bootstrap ช่วยให้การจัดวางองค์ประกอบบนหน้าเว็บมีความยืดหยุ่นและตอบสนองต่อขนาดหน้าจอต่างๆ ได้อย่างมีประสิทธิภาพ ตัวอย่างการใช้งาน: HTML
